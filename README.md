# Dress Visualizer

For setting up and running project from scratch

Step 1- Download the zip file of the project and extract it somewhere in your system

Step 2- Open the extracted file in VS Code

Step 3- Now open the terminal and run "npm install"(Thiss will install all the dependecies)

Step 4- After that run "npm start" to run it on you local host


How to use it:-

Step 1- Upload the human image and the fabric or cloth image

Step 2- Click on image processing

Step 3- You will see the human image and the fabric both on the canvas whre you can keep the fabric as you want on the human image 

Step 4- After all the changes on canvas click Submit

Step 5- Now you can see the final processed image in the processed image section
